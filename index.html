Виды селекторов

Варианты

  • * - Универсальный селектор соответствует любому элементу на странице.
  • #X - Использование символа решётки позволяет обратиться к уникальному идентификатору элемента id.
  • .X - class - это селектор класса. Отличие селектора class от id в том, что класс не должен быть уникальным и может использоваться для выбора нескольких элементов на странице.
  • X Y - Ещё один из часто используемых типов селекторов - это контекстный селектор. Он используется для выбора элементов, удовлетворяющих определённому контексту.
  • X (tag) - В случае если вы не можете использовать id или class можно обратиться напрямую к тегу.
  • X:visited и X:link - Мы используем псевдокласс :link для стилизации ссылок, на которые пользователь ещё не нажимал. Также нам доступен псевдокласс :visited для стилизации только тех ссылок, на которые пользователь нажал.
  • X + Y - Это соседний селектор. Он поможет нам выбрать только тот элемент, который следует сразу же за указанным элементом.
  • X > Y - Разница между селекторами X Y и X > Y в том, что в последнем примере мы выберем только прямого потомка.
  • X[title] - Это селектор атрибутов.
  • X[href*="foo"] - Селектор который учитывает значение ссылки
